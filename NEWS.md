# DeIDmachinedata 0.4.0

## New features

* Added device-specific wrapper functions:
  - deidentify.spectralis.raw()
  - deidentify.spectralis.pdf()
  - deidentify.cirrus()
  - deidentify.hfa()

* Added general-purpose deidentify.metadata() engine.

* Improved input validation and overwrite handling.

* Extended date shifting mechanism to support epoch seconds.
